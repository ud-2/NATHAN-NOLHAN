# INFO - QUICK TEST 1

**Instructions:** Write a short, complete Python script or function for each of the 10 prompts below. 

### Section 1: Basic I/O, Types, and Math (Questions 1-4)
- **1.** Write a program that asks the user for their age and prints their year of birth. *(Assume the current year is 2026).*
- **2.** Write a program that asks the user for a number of days, then calculates and prints how many full weeks and remaining days that represents (e.g., 10 days = 1 week and 3 days).
- **3.** Write a program that asks the user for the price of an item. Apply a 15% discount to this price and print the new final price.
- **4.** Write a program that asks for two numbers and prints their absolute difference. **Constraint:** You must use an `if/else` statement. You are *not* allowed to use the built-in `abs()` function.

### Section 2: Loops and Conditions (Questions 5-10)
- **5.** Write a `while` loop that prints all the **even** numbers strictly descending from 20 down to 2.
- **6.** Write a `for` loop that calculates and prints the total sum of all integers from 1 to 100 (inclusive).
- **7.** Write a program that asks the user to input a word. Use a loop to print each letter of the word on a new line. However, if the loop encounters the letter `'x'`, it must stop printing immediately (break the loop).
- **8.** Write a program that prints the multiples of 5, from 5 up to 50, separated by commas.
- **9.** Write a program that asks the user for a number `n`. Print "Fizz" if the number is a multiple of 3, "Buzz" if it is a multiple of 5, and the number itself if it is neither.
- **10.** Write a loop that asks the user to enter a password. The loop should keep asking "Enter password:" until the user types the exact word `"secret"`. Once they do, print "Access Granted".
